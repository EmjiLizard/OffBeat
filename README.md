# OffBeat

<div>
<h1>OffBeat</h1>
<p>OffBeat is a free, offline music player app intended to provide playlist and shuffle functionality for local music files. The idea is to create an option-even if small-other than streaming and to facilitate a world where listeners actually <i>own</i> their music.</p>
\n
<p>Currently OffBeat is being designed towards an initial iOS release, and to support MP3 and .WAV files, although eventual support for other platforms and formats are planned.</p>
